# rubocop-redis

An extension of RuboCop for [redis/redis-rb](https://github.com/redis/redis-rb) Redis client.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rubocop-redis'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install rubocop-redis
